## 衛星影像 API

| API 名稱 | 功能與用途 | 免費性與限制 | 回應格式/調用方式 | 回應速度與延遲 | 整合考量 | API Key/文件支持 |
| --- | --- | --- | --- | --- | --- | --- |
| **NASA 地球影像 API**(Earth Imagery) | 提供特定地點的衛星照片（Landsat 8 衛星影像）。開發者可依據經緯度和日期獲取地球指定區域的對應衛星圖像，用於即時地球觀測展示或背景紋理[github.com](https://github.com/nasa/earth-imagery-api#:~:text=This%20repository%20contains%20a%20flask,product%2C%20Landsat%208%20collection%202)。例如可取得某地近期的衛星雲圖，用於氣象或地理應用 | 完全免費（NASA 政策資料開放）但**需要申請 API Key**。使用預設的 API Key 每小時可達 1,000 次請求[apidog.com](https://apidog.com/blog/how-to-use-nasa-api/#:~:text=1,programming%20techniques%20to%20improve%20performance)；若無提供 Key 則使用示範 Key（配額更低）。由於服務主要供教學與小型項目，不建議大量商業調用[github.com](https://github.com/nasa/earth-imagery-api#:~:text=,Sourced) | 透過 RESTful GET 請求獲取資料：提供經緯度、日期等參數給 `planetary/earth/imagery` 端點即可。回應可為影像檔案（JPEG/PNG）或包含影像URL的 JSON；相關輔助端點如 `assets` 可列出可用影像日期[github.com](https://github.com/nasa/earth-imagery-api#:~:text=This%20repository%20contains%20a%20flask,product%2C%20Landsat%208%20collection%202) | 請求小型縮略影像時延遲低（數百毫秒級）；但若請求高解析度影像，取決於檔案大小（數MB時可能需要數秒下載）。整體服務可靠性尚可，但偶爾可能出現500錯誤須重試[github.com](https://github.com/nasa/earth-imagery-api#:~:text=,Sourced)。不適合高頻率連續抓圖（NASA 建議僅作示範用途） | 後端FastAPI可調用此API下載影像並緩存（例如每日自動抓取最新影像）。由於影像為二進制資料，後端可轉為Base64或直接透傳給前端。可將取得的地球照片作為Three.js地球模型的貼圖，以供前端實時呈現地球外觀變化。但需注意避免頻繁請求，可結合快取與定時更新 | **需要 API Key**（免費申請）[apidog.com](https://apidog.com/blog/how-to-use-nasa-api/#:~:text=1,programming%20techniques%20to%20improve%20performance)；NASA 官方文件詳細列出參數選項（例如`lat`、`lon`、`date`）和使用說明。整合門檻低，只需一般HTTP請求，但請遵守NASA用量指引[github.com](https://github.com/nasa/earth-imagery-api#:~:text=This%20repository%20contains%20a%20flask,product%2C%20Landsat%208%20collection%202)[github.com](https://github.com/nasa/earth-imagery-api#:~:text=,Sourced) |
| **NASA EPIC API**(地球全照影像) | 提供深空氣候觀測衛星 DSCOVR 的地球多彩相機（EPIC）每日拍攝的**地球全盤影像**資料[wilsjame.github.io](https://wilsjame.github.io/how-to-nasa/#:~:text=,event%20metadata%2C%20such%20as%20storm)。可取得每天多次的地球全光照圖像（天然色或增強色），以及每張影像的時間戳與元數據。適合用於即時地球可視化（如展示地球白天側的變化）或科學教育 | 免費提供，需使用 NASA API Key（同屬 api.nasa.gov 平台）。每小時調用上限與其他 NASA API 相同（約 1000 次）[apidog.com](https://apidog.com/blog/how-to-use-nasa-api/#:~:text=1,programming%20techniques%20to%20improve%20performance)。由於每天影像數量有限（通常10-20張），一般應用不易觸及額度限制 | 通過 RESTful 請求獲取 JSON 資料列表，例如 `EPIC/api/natural` 可取得最近幾張影像的資訊，或 `.../date/YYYY-MM-DD` 獲取特定日期的所有影像列表[proulxp.github.io](https://proulxp.github.io/CS290-How-To-Guide/epic.html#:~:text=var%20exampleURL%20%3D%20%22https%3A%2F%2Fapi.nasa.gov%2FEPIC%2Fapi%2Fnatural%2Fdate%2F2015)[proulxp.github.io](https://proulxp.github.io/CS290-How-To-Guide/epic.html#:~:text=The%20key%20concept%20here%20is,31)。回應 JSON 包含每張影像的文件名、拍攝時間、地球太陽角度等；需根據文件名再組合URL下載實際影像JPEG檔 | JSON 資料本身體積小，延遲低（<0.5秒）。下載實際影像取決於大小（每張約2～5MB）；由於每天影像數量有限，整體流量不大。DSCOVR 約每隔2小時拍攝一次，全盤影像更新延遲在影像拍攝後數小時內發布，對於實時性要求不是秒級 | FastAPI 可定時抓取最新 EPIC 影像列表，將URL提供給前端。由於需要組裝影像URL並下載，可在後端完成後再傳給前端（減少前端負擔）。在 React+Three.js 前端中，可將最新的地球全盤照片貼圖到3D地球球體上，實現接近即時的地球視圖；或播放連續影像形成地球自轉動畫。注意儲存影像以避免重複下載 | **需要 API Key**；NASA 提供官方文件說明 EPIC API 的使用方式，包括如何查詢指定日期及取得影像下載連結[proulxp.github.io](https://proulxp.github.io/CS290-How-To-Guide/epic.html#:~:text=The%20first%20specific%20API%20we,API%20can%20be%20found%20here)[proulxp.github.io](https://proulxp.github.io/CS290-How-To-Guide/epic.html#:~:text=The%20key%20concept%20here%20is,31)。整合需處理影像下載與緩存，門檻中等 |

## 整合建議與應用情境

綜合上述 API，可以打造各種太空主題的互動應用。在整合時建議由 **FastAPI 後端**統一調度這些 API，妥善緩存資料並提供簡化的接口給前端，確保流暢且不超限。前端（例如 React 結合 Three.js）則專注於即時視覺化呈現。以下是幾個潛在的應用場景：

- **國際太空站追蹤看板**：利用 ISS 即時位置 API 搭配 Three.js 的地球模型，實現 ISS 围繞地球的即時軌跡展示。後端定期調用 Open Notify 或 Where the ISS at API 獲取最新座標，高度等資訊，前端渲染一個繞地球運動的 ISS 模型，並疊加 NASA 衛星影像作為地球貼圖，提升逼真度。可加入 Spaceflight News API 提供的即時新聞，比如當前 ISS 任務更新或宇航員相關消息，在界面側欄滾動顯示，增強資訊性。
- **天文事件互動地圖**：整合 USNO 天文資料與 Arcsecond API，製作一個可查詢天文現象的互動星空應用。使用後端計算特定地點今晚的日落時間、月出方位，或下一次日食發生日期。Arcsecond 提供行星當前天空位置，用 Three.js 繪製出當前行星和星座在天空中的位置示意。使用者可在前端選擇日期/地點，FastAPI 調用API計算結果並返回給前端即時更新星空圖。這對天文愛好者或教育演示非常有幫助。
    
    [aa.usno.navy.mil](https://aa.usno.navy.mil/data/api#:~:text=,Earth%27s%20Seasons%20and%20Apsides)
    
- **地球即時觀測與新聞介面**：結合 NASA EPIC 每日地球全照影像，製作一個「今日地球」展示頁，每隔幾小時更新地球全貌圖像（由後端抓取最新EPIC影像）。同時調用 Spaceflight News API 獲取最新的太空探索新聞列表，與地球圖像並列展示。使用者可以旋轉地球查看不同區域的日夜變化，同步閱讀相關的太空新聞。FastAPI 在此負責定時下載最新地球影像並提供給前端（或直接讓前端透過 URL 加載），以及聚合新聞數據。整體提供一個即時且富媒體的太空資訊體驗。

上述整合應用透過 FastAPI **有效管理各API的調用**，避免前端直接訪問造成的重複請求或密鑰洩露。同時利用緩存策略提升響應速度（例如發射事件列表每小時更新一次，新聞每10分鐘抓取一次），前端則透過 WebSocket 或輪詢獲取更新，在不影響性能的前提下實現**即時互動**。這種架構能充分發揮各免費太空 API 的優勢，在零成本的情況下打造出豐富的太空資料可視化和資訊平台。每個 API 的官方文件都相當完善，上手門檻不高，開發者可以依據應用需要靈活組合這些資源。總之，藉由合理的服務調度與前後端配合，這些免費太空 API 足以支撐起功能強大的太空資訊應用。


# 台灣太空資料開放 API 一覽

以下整理多項與台灣太空相關的開放 API，涵蓋衛星影像、太空任務數據、天文觀測、航太研究以及氣象與地球觀測資料等領域。每項 API 列出提供單位、名稱與連結、資料內容功能、格式、使用授權限制，以及適用場景，以供專案應用參考。

## 開放 API 清單

### 國家太空中心（TASA）- 福衛二號衛星影像 WMTS 服務

- **提供單位：** 國家太空中心 (Taiwan Space Agency, TASA)，由中研院 GIS 平臺協助提供
- **API 名稱與連結：** 福爾摩沙衛星二號無雲全台正射影像 WMTS 服務（Web Map Tile Service）。服務入口為 `gis.sinica.edu.tw/fs-2/`（WMTS GetCapabilities) ([QGIS及Open Geodata資源網@Sinica » 福衛二號影像WMTS服務新增2014及2015年份影像](https://gis.rchss.sinica.edu.tw/qgis/archives/3445#:~:text=%E7%A6%8F%E8%A1%9B%E4%BA%8C%E8%99%9F%E5%BD%B1%E5%83%8FWMTS%E6%9C%8D%E5%8B%99%EF%BC%88http%3A%2F%2Fgis.sinica.edu.tw%2Ffs))
- **資料內容與功能：** 提供2005年至2015年福爾摩沙衛星二號（福衛二號）拍攝的全台灣2公尺解析度無雲遙測影像年度鑲嵌圖層，共11年之衛星影像 ([QGIS及Open Geodata資源網@Sinica » 福衛二號影像WMTS服務新增2014及2015年份影像](https://gis.rchss.sinica.edu.tw/qgis/archives/3445#:~:text=%E7%A6%8F%E8%A1%9B%E4%BA%8C%E8%99%9F%E5%BD%B1%E5%83%8FWMTS%E6%9C%8D%E5%8B%99%EF%BC%88http%3A%2F%2Fgis.sinica.edu.tw%2Ffs))。使用者可透過 WMTS 介接取得地圖磚影像，用於製作地圖疊圖、遙測分析等
- **資料格式：** WMTS 服務以地圖磚影像形式提供（如 JPG/PNG 影像）。符合 OGC WMTS 標準，可透過地理座標與縮放級別請求影像
- **使用授權與限制：** 開放使用，**無需 API 金鑰或註冊**。屬於政府開放資料，供大眾自由取用（僅需遵守資料來源標示等開放資料一般授權規範）
- **適用的應用場景：** 可應用於地理資訊系統（GIS）、國土測繪、環境監測、災害評估等，需要高解析度遙測底圖的情境。例如製作台灣地表變遷地圖、分析颱風災後地貌變化等

### 國家太空中心（TASA）- 福衛七號/COSMIC-2 太空任務資料 (TACC 平臺)

- **資料內容與功能：** 提供福衛七號/COSMIC-2 衛星任務的**GNSS掩星觀測資料**，包括中性大氣層折射率廓線、電離層全電子含量、掩星訊號相位延遲等多種衍生產品。資料依不同處理級分為大氣折射率/溫度剖面、電離層參數、衛星軌道與原始觀測值等，涵蓋即時資料與事後處理資料，可供科學研究與天氣預報使用 ( [](https://tacc.cwa.gov.tw/v2/download.html#:~:text=)) ( [](https://tacc.cwa.gov.tw/v2/download.html#:~:text=))。例如提供大氣折射率剖面 (atmPrf)、電離層電子密度 (podTc2) 等檔案供下載分析

- **使用授權與限制：** **免費開放取用**。福衛七號/COSMIC-2 資料遵循國際自由開放資料政策，任何人皆可依「資料釋出備忘錄」使用 ([[PDF] Status and Aims of the FORMOSAT-7 / COSMIC-2 Mission - ECMWF](https://www.ecmwf.int/sites/default/files/elibrary/2014/13747-status-and-aims-formosat-7-cosmic-2-mission.pdf#:~:text=ECMWF%20www.ecmwf.int%20%20Following%20FORMOSAT,data%20policy%20to%20provide))。無需註冊即可下載，但要求使用者閱讀並遵守資料使用需知（如引用來源等） ( [](https://tacc.cwa.gov.tw/v2/download.html#:~:text=FORMOSAT,before%20using%20this%20data))。
- **適用的應用場景：** **氣象預報**（數值天氣預報同化掩星大氣剖面改善預報準確度）、**氣候研究**（分析全球大氣折射率變化）、**電離層與太空天氣監測**（利用電離層電子含量資料研究太空天氣對通訊導航之影響）、以及**學術研究**（大氣科學、地球科學領域教研）等。由於資料全球覆蓋且即時性強，也適用於國際天氣預報中心和學術單位進行大氣監測 ([[PDF] Status and Aims of the FORMOSAT-7 / COSMIC-2 Mission - ECMWF](https://www.ecmwf.int/sites/default/files/elibrary/2014/13747-status-and-aims-formosat-7-cosmic-2-mission.pdf#:~:text=ECMWF%20www.ecmwf.int%20%20Following%20FORMOSAT,data%20policy%20to%20provide))

### 農業部漁業署 - 衛星海面溫度影像資料 API

- **提供單位：** 農業部漁業署（原農委會漁業署）
- **API 名稱與連結：** 衛星海表面溫度影像資料開放介接。透過農業開放資料平臺提供 REST 查詢：例如 `data.moa.gov.tw/Service/OpenData/FromM/SatelliteImageSASC.aspx` 等，可取得最新的海溫影像資訊 ([Satellite sea surface temperature image ｜ 政府資料開放平臺](https://data.gov.tw/en/datasets/9810#:~:text=Resources%20download%20link))。（也可經由政府資料開放平臺找到**「衛星海面溫度影像」**資料集）
- **資料內容與功能：** 提供每日更新的**海表面溫度衛星觀測圖**。資料包含日期及該日對應的**衛星海溫圖** ([Satellite sea surface temperature image ｜ 政府資料開放平臺](https://data.gov.tw/en/datasets/9810#:~:text=Satellite%20sea%20surface%20temperature%20image))。使用者可取得當天由衛星遙測之海面溫度分布影像，用於海洋環境監測。這套API每天發布最新圖像，讓應用程式可及時抓取海溫變化情形（如黑潮位置、海表暖冷流）
- **資料格式：** 支援多種格式輸出，例如 **JSON、CSV、XML** 等 ([Satellite sea surface temperature image ｜ 政府資料開放平臺](https://data.gov.tw/en/datasets/9810#:~:text=Resources%20download%20link))。JSON/XML 格式下提供欄位如日期和對應影像網址；CSV 則列出日期和影像連結，以供批次下載或資料庫匯入。影像本身通常為圖檔（如PNG/JPEG）透過上述連結取得
- **使用授權與限制：** 採用**政府資料開放授權條款1.0**，免費公開提供 ([Satellite sea surface temperature image ｜ 政府資料開放平臺](https://data.gov.tw/en/datasets/9810#:~:text=License))。**無需註冊 API Key**，任何人皆可直接透過網址介接取得資料。使用時僅需遵守開放資料使用規範（如標明來源：農業部漁業署）即可
- **適用的應用場景：** **漁業與海洋預報**（如漁場分析、掌握漁場水溫）、**海洋環境監測**（觀測黑潮等洋流動態、海洋熱浪事件）、**氣候研究**（研究海表溫度與氣候現象的關聯，如聖嬰現象）、**教育展示**（製作海洋科學教材）等需要海表溫度資料的情境

### 政府資料開放平臺 (data.gov.tw) - 開放資料入口網站

- **提供單位：** 國家發展委員會 (NDC) 主導維護，彙整各政府機關/單位資料
- **API 名稱與連結：** 政府資料開放平臺 (data.gov.tw) 提供資料集搜尋及下載介面。網站本身提供 [資料集查詢介面](https://data.gov.tw/) ，部分資料集也提供 API 介接或下載連結。另有 CKAN 平臺 API 供開發者搜尋資料集資訊
- **資料內容與功能：** 平臺彙集了**全國中央及地方機關的開放資料**，涵蓋主題廣泛，包括上述太空相關資料（氣象、衛星）、交通、環境、教育、文化等數萬筆資料集。開發者可透過關鍵字或主題分類搜尋所需資料集，取得其下載連結或API端點 ([Chii-Bin Wu - HOWTO-CWB-1](https://sites.google.com/view/chiibinwu/teaching/softwares/python/python-tutorial/howto-cwb-1#:~:text=%E2%80%BB%E5%B0%87%E4%B8%AD%E6%96%87%E8%BD%89%E6%8F%9B%E7%A8%8B%E7%99%BE%E5%88%86%E6%AF%94%E7%AC%A6%E8%99%9F%E7%9A%84ASCII%E7%A2%BC%EF%BC%9A%20https%3A%2F%2Fwww.url))。平臺定期公告新的資料集上架訊息，提供資料描述、欄位說明及授權方式，方便使用者瞭解資料內容
- **資料格式：** 視各資料集而定，常見格式如 **JSON、XML、CSV、GeoJSON、KML、Shapefile、GeoTIFF** 等均有涵蓋。平臺本身亦提供資料集目錄的機器可讀介面（如 CKAN API，以 JSON 提供資料集清單）。因此開發者能取得結構化的元數據，再依資料集提供的資源連結獲取特定格式檔案
- **使用授權與限制：** 大部分資料集採用**開放政府資料授權條款1.0** ([Satellite sea surface temperature image ｜ 政府資料開放平臺](https://data.gov.tw/en/datasets/9810#:~:text=License))或相容之開放授權，免費供使用。**無統一授權碼機制**：除非資料提供單位另有要求（如上述氣象署資料須申請金鑰），一般資料可直接下載使用。使用者需遵守各資料集標示之授權條款，通常要求標示來源，允許商業利用但不負擔資料正確性等
- **適用的應用場景：** 作為**開放資料的一站式入口**，適用於各類專案尋找所需資料。例如想取得**太空領域相關**開放資料時，可在此平臺搜尋「衛星」「天文」「氣象」等關鍵字快速定位資料集。無論是開發手機應用、進行資料分析研究，或學術單位彙整資料，皆可利用此平臺方便地取得台灣各政府單位釋出的開放資料 ([Chii-Bin Wu - HOWTO-CWB-1](https://sites.google.com/view/chiibinwu/teaching/softwares/python/python-tutorial/howto-cwb-1#:~:text=%E2%80%BB%E5%B0%87%E4%B8%AD%E6%96%87%E8%BD%89%E6%8F%9B%E7%A8%8B%E7%99%BE%E5%88%86%E6%AF%94%E7%AC%A6%E8%99%9F%E7%9A%84ASCII%E7%A2%BC%EF%BC%9A%20https%3A%2F%2Fwww.url))
